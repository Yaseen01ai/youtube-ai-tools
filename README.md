# Creator AI Tools

Open-source AI tools for creators, focused on content workflow, title generation, and productivity.

## Features
- Generate video title ideas
- Brainstorm content hooks
- Assist with story-based content workflows
- Simple automation utilities for creators

## Purpose
This project is an experimental open-source toolkit for creators who want faster ideation and lightweight AI-assisted workflows.

## Roadmap
- Title generator
- Hook generator
- Thumbnail text helper
- Story prompt generator
